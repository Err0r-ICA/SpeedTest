# SpeedTest - Test your bandwidth Download/Upload speed

## Installation:
```
pkg update -y;pkg upgrade -y;git clone https://github.com/Err0r-ICA/SpeedTest;cd SpeedTest;pip install -r requirements.txt
python2 Sp33d
```
